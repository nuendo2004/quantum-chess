# Quantum Play: CS555 PROJECT

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

The first time running, in shell enter:

```bash
npm install
```

To run tests, enter:

```bash
npm run test
```

#### setup

Create a .env.local file in root folder, add all requird env variables.
